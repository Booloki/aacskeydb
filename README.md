aacskeydb
=============

aacs keys database downloader is a workaround to play encrypted Blu-ray Discs

About
-------

It only download KEYDB.cfg from [vlc-bluray.whoknowsmy.name](http://vlc-bluray.whoknowsmy.name/files/) to ~/.config/aacs/
